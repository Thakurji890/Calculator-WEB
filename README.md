# Basic Calculator Web App

A simple **online calculator** built using **HTML, CSS, and JavaScript** that performs basic mathematical operations such as addition, subtraction, multiplication, and division.

## Features

- Perform basic arithmetic operations
  - Addition (+)
  - Subtraction (-)
  - Multiplication (\*)
  - Division (/)

- Clear display functionality
- Delete last entered value
- Responsive and simple user interface
- Error handling for invalid expressions

## Technologies Used

- **HTML** – Structure of the calculator
- **CSS** – Styling and layout
- **JavaScript** – Calculator logic and functionality

## Project Structure

```
calculator-project
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## How It Works

1. The user clicks number and operator buttons.
2. The values are displayed in the calculator screen.
3. When the **"="** button is pressed, JavaScript evaluates the expression.
4. The result is displayed on the screen.

## How to Run the Project

1. Download or clone the repository

```
git clone https://github.com/Thakurji890/calculator-project.git
```

2. Open the project folder.

3. Run the project by opening **index.html** in any web browser.

## Example

```
Input: 5 + 3 * 2
Output: 11
```
